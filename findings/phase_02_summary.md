# phase 02 summary

objective:

evaluate symbolic reasoning robustness under graph corruption.

experiments:

- noisy graph generation
- path deviation analysis
- hallucination detection
- multi-seed robustness evaluation

results:

- path change rate: 50%
- success-to-failure transitions observed
- hallucination-like shortcuts detected
- stable, degraded, and collapsed reasoning regimes identified

conclusion:

graph structure quality directly impacts symbolic reasoning performance.

reasoning correctness and reasoning faithfulness should be evaluated independently.
